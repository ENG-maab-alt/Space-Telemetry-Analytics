# Space-Telemetry-Analytics

End-to-end data engineering project analyzing real-world spacecraft telemetry data using Python and Power BI.

## 🚀 Project Overview
This project focuses on the anomaly detection of spacecraft telemetry data, utilizing Python for exploratory data analysis (EDA) and Power BI for interactive visualization.

## 📊 Methodology
1. **Data Preparation**: Cleaning and processing raw sensor logs.
2. **Exploratory Analysis**: Using Python (Matplotlib) to identify signal spikes and outliers.
3. **Dashboard Design**: Built a hierarchical Power BI dashboard to monitor spacecraft health.

## 📈 Key Findings
- **SMAP Dominance**: Identified that SMAP accounts for 85.76% of detected anomalies.
- **Anomaly Clustering**: Discovered specific event sequences where failure rates peak.
- **Predictive Insight**: Recommend implementing threshold-based alerts to mitigate potential failures.

## 🛠 Tech Stack
- **Python**: Data preprocessing and EDA.
- **Power BI**: Data visualization and interactive reporting.
- **Git/GitHub**: Version control and project documentation.

- Developed skills in cleaning noisy telemetry data.
